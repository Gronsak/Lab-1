# Lab 1
This code is meant to be used to solve a lab for a course on programming.
It calls an api to get a token then verifys the token by fetching a secret and finally exchanges the token and secret for a flag that marks that the task has been completed succesfylly.

# Parameters and flags
- -h, --help: Show a help message
- -u, --url: Set the API server URL to use (-u "http://www.example.com)
